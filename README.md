# Portafolio de Ciberseguridad — Julian Eduardo

Proyectos prácticos de ciberseguridad desarrollados para demostrar habilidades en detección de amenazas, análisis de logs y automatización de seguridad.

## Proyectos

### 🔐 [Mini-SIEM](./mini-siem/)
Sistema de gestión de eventos de seguridad desarrollado en Python. Recolecta eventos del Event Log de Windows, detecta patrones de ataque (fuerza bruta, escalación de privilegios, persistencia) y los visualiza en un dashboard web en tiempo real.

**Stack:** Python · Flask · SQLite · pywin32
